# Gap Analysis: Animation Assistant vs Professional Curricula

## Methodology
Compared our 18 existing modules against:
- Professional animation curricula (SVA BFA, CGSpectrum, Animation Mentor)
- Industry skills lists (Lumen & Forge "Top Skills Animators Need 2025")
- Community guides (Reddit r/animation, Animator Island)
- Animation FAQs (Animaker, Austin Visuals, Animator Island Q&A)
- Raspberry Pi animation resources (Raspberry Pi Foundation projects, TutsPlus, MakeUseOf, Doctor Geek/Nerd)
- Animation portfolio/career advice (CGSpectrum, Creative Assembly, Reddit r/animationcareer)

## GAPS IDENTIFIED — 10 Missing Areas

### Gap 1: Color Theory & Visual Design
**Why it matters:** Professional curricula consistently list color theory, composition, perspective drawing, and background/environment design as core artistic skills. We teach staging (where to look) but not HOW scenes look — color mood, composition rules, visual hierarchy, perspective for depth.
**Sources:** Lumen & Forge skills list; SVA BFA curriculum; Chris Brejon CG cinematography

### Gap 2: Sound & Audio Design for Animation
**Why it matters:** Sound is 50% of animation. We mention audio import in tool tips but have no module on sound design — SFX timing, music selection, voice recording, syncing audio to action, layering ambient sound. Kids' animations feel empty without sound guidance.
**Sources:** Animaker FAQ; Animation Mentor lip sync tutorial; general animation pedagogy

### Gap 3: Rigging Deep Dive
**Why it matters:** Rigging is listed as a core technical skill across all professional sources. We define it in the glossary and mention it in tool comparison, but have no teaching module for kids ready to move from frame-by-frame to rigged/puppet animation. This is the bridge from beginner to intermediate.
**Sources:** Lumen & Forge; CGSpectrum; Synfig/OpenToonz/Blender documentation

### Gap 4: Compositing & Rendering
**Why it matters:** Compositing, video editing, and rendering are listed as essential technical skills. Kids can animate but don't know how to layer elements, add effects, composite multiple passes, or properly render final output. This is the "make it look finished" skill.
**Sources:** Lumen & Forge; Gnomon Workshop; industry workflows

### Gap 5: Portfolio & Career Path
**Why it matters:** Multiple sources emphasize portfolio/demo reel as more important than degrees. Teens 14-17 need guidance on building portfolios, creating demo reels, understanding career paths, and what the industry looks for. Our showcase module (12) is about celebrating work; this is about career preparation.
**Sources:** CGSpectrum portfolio tips; Reddit r/animationcareer; Creative Assembly showreel advice; Instagram animator advice

### Gap 6: Animation Styles & Techniques Guide
**Why it matters:** The Animaker FAQ identifies many animation types kids should know about: rotoscope, cut-out, claymation, cel animation, morphing, motion graphics. Kids may discover they love a style they've never heard of. We cover tools but not the distinct animation techniques/styles themselves.
**Sources:** Animaker FAQ; animation style discussions; Reddit communities

### Gap 7: Collaboration & Team Animation
**Why it matters:** Professional animation is collaborative (Pixar's "Braintrust" model, team workflows). Kids in classrooms, clubs, or online communities need to know how to split work, review each other's animation, and build together. No module addresses this.
**Sources:** Lumen & Forge (communication & teamwork); Pixar Creativity Inc. model; school animation clubs

### Gap 8: Raspberry Pi & Low-Resource Animation
**Why it matters:** The Raspberry Pi Foundation has dedicated animation projects (push-button stop motion with Pi Camera + Python + GPIO). The `stopmotion` apt package, `picamzero` library, `ffmpeg` for video assembly, and Scratch/Pygame all run on Pi. Kids with a Pi (or limited hardware) need a dedicated guide. This is an equity/access issue.
**Sources:** Raspberry Pi Foundation projects; TutsPlus stop motion studio; MakeUseOf Pi rig; Doctor Geek/Nerd blog; raspberrypi.stackexchange

### Gap 9: Kid-Friendly FAQ
**Why it matters:** FAQs from Animaker, Austin Visuals, and Animator Island reveal common questions kids actually ask: "What was the first animation?", "Do I need to be good at drawing?", "Can I animate on a phone?", "How long does it take?", "Can I make money?", "What computer do I need?", "What's the difference between anime and animation?" We have a glossary (terms) and history (facts) but no FAQ (answers to common questions).
**Sources:** Animaker FAQ; Austin Visuals FAQ; Animator Island Q&A; Reddit r/animation FAQs

### Gap 10: Acting for Animation
**Why it matters:** Multiple professional sources state "the #1 skill is acting." Understanding performance, emotion through body language, facial expressions, and character motivation separates technical animators from great animators. We reference emotion in projects (06, 16) but have no dedicated module on acting/performance for animation.
**Sources:** Quora animator skills; Animation Mentor; CGSpectrum; professional animator interviews
