# Animation Tools Landscape for Ages 10-17

## Learning Progression Path

```
Beginner (10-12)          Intermediate (12-14)         Advanced (14-17)
─────────────────────────────────────────────────────────────────────
Scratch ──────────────► Krita ──────────────────► OpenToonz
Stop Motion Studio ──► Pencil2D ───────────────► Blender (3D)
                        Synfig (tweening) ──────► Blender (Grease Pencil)
```

## Tool Profiles

### Scratch (scratch.mit.edu)
- **Type:** Block-based visual programming
- **Cost:** Free, web-based
- **Best for:** Ages 10-13, absolute beginners
- **Strengths:** Sprite-based, costumes, broadcast messages, story projects
- **Weakness:** Not "real" animation software — more programming-driven
- **Key concepts to teach:** Frames via costume changes, timing via wait blocks, loops, broadcast coordination

### Stop Motion Studio
- **Type:** Stop-motion capture (phone/tablet/webcam)
- **Cost:** Free / $5 pro upgrade
- **Best for:** Ages 10-14, tactile learners
- **Strengths:** Onion skinning, frame-by-frame, works with LEGO/clay/drawings
- **Weakness:** Not digital drawing — needs physical setup
- **Key concepts to teach:** Physical animation principles, frame rate, timing, easing

### Pencil2D (pencil2d.org)
- **Type:** 2D hand-drawn animation
- **Cost:** Free, open-source
- **Best for:** Ages 10-15, first digital animation
- **Strengths:** Minimal interface, intuitive, raster + vector layers
- **Weakness:** No tweening, limited features beyond frame-by-frame
- **Key concepts to teach:** Onion skinning, frame-by-frame, keyframes, timing

### Krita (krita.org)
- **Type:** Digital painting + animation
- **Cost:** Free, open-source
- **Best for:** Ages 12-16, kids who already like to draw
- **Strengths:** Excellent drawing tools, onion skin docker, audio import
- **Weakness:** Animation features secondary to painting — not a full animation suite
- **Key concepts to teach:** Onion skin, timeline, frames, export to video

### Synfig Studio (synfig.org)
- **Type:** 2D vector tweening animation
- **Cost:** Free, open-source
- **Best for:** Ages 12-16, kids who don't want to draw every frame
- **Strengths:** Tweening, rigging, puppet animation, vector-based
- **Weakness:** Interface is "cumbersome and obtuse" per community consensus
- **Key concepts to teach:** Keyframes, interpolation, tweening, rigging basics

### OpenToonz (opentoonz.org)
- **Type:** 2D professional animation
- **Cost:** Free, open-source (Studio Ghibli's tool)
- **Best for:** Ages 14-17, serious students
- **Strengths:** Frame-by-frame + tweening, multiplane camera, professional features
- **Weakness:** Steep learning curve, complex interface
- **Key concepts to teach:** Full 2D pipeline, scene planning, camera moves, lip sync

### Blender (blender.org)
- **Type:** 3D + 2D (Grease Pencil)
- **Cost:** Free, open-source
- **Best for:** Ages 14-17, ambitious students
- **Strengths:** Full 3D pipeline, Grease Pencil for 2D/2.5D, massive community
- **Weakness:** Very complex, can overwhelm beginners
- **Key concepts to teach:** 3D keyframing, Grease Pencil 2D, rigging, rendering

## The 12 Principles of Animation (Teaching Framework)

1.  **Squash & Stretch** — Shape change to show weight/flexibility
2.  **Anticipation** — Wind-up before action
3.  **Staging** — Clear focal point for viewer
4.  **Straight Ahead vs. Pose to Pose** — Two approaches to animation
5.  **Follow Through & Overlapping Action** — Parts don't stop together
6.  **Slow In & Slow Out** — Easing (most important for digital tools)
7.  **Arcs** — Natural movement follows curved paths
8.  **Secondary Action** — Supporting movements that enrich
9.  **Timing** — Frames per action = speed/weight
10. **Exaggeration** — Push beyond realism for clarity
11. **Solid Drawing** — 3D weight, form, balance
12. **Appeal** — Character/design that's interesting to look at

## Recommended Project Progression for Kids

1.  **Bouncing ball** (squash & stretch, timing, arcs)
2.  **Walking character** (follow-through, overlapping action, timing)
3.  **Simple face turn** (anticipation, slow in/out)
4.  **Character jump** (anticipation, squash & stretch, follow-through)
5.  **Story scene** (staging, secondary action, appeal)
6.  **Lip-sync dialogue** (timing, solid drawing)
7.  **Full short film** (all principles combined)
